# Sharad.

**Self-taught builder designing minimalist macOS utilities, privacy-first systems, and tools I wish existed.**

---

I build software driven by curiosity, high-fidelity design, and the utility of independent tools. My focus is on native desktop experiences, high-performance backend pipelines, and running self-hosted, decentralized infrastructure.

Based in India. Shipped apps in Swift, backend APIs in Rust/Go, and interactive frontends in Next.js.

---

### 🛠️ Active Focus Areas
*   **Desktop Engineering** — Crafting rich macOS native environments using **Swift**, **SwiftUI**, and **SwiftData**.
*   **Distributed Systems** — Building high-performance network services in **Rust** and cloud-native backends in **Go**.
*   **Local AI** — Devising offline-first desktop workflows powered by lightweight open-source LLMs run locally via Ollama.

---

### 📂 Featured Open-Source Projects

#### 📱 Desktop & Utilities
*   **[ziro](https://github.com/bysharad/ziro)** — An anime-inspired macOS productivity dashboard featuring focus timers, habit trackers, task managers, and local audio/video libraries. Built natively with **SwiftUI**, **SwiftData**, and **AVFoundation**.
*   **[PomodoroTimer](https://github.com/bysharad/PomodoroTimer)** — A premium, minimalist menu-bar Pomodoro timer featuring anime GIF integrations, detailed session statistics, and a sleek black-and-white theme. Built with **SwiftUI**.

#### ⚙️ Systems & Backends
*   **[novaseek](https://github.com/bysharad/novaseek)** — A privacy-first, independent web search engine and concurrent crawler that queries and indexes pages with zero user-data collection. Written from scratch in **Rust** using **Axum** and **Tantivy**.
*   **[cloudmusic](https://github.com/bysharad/cloudmusic)** — A Subsonic-compatible music cataloging server that connects to user-owned storage, indexes audio metadata, and exposes a high-fidelity streaming interface. Powered by **Go**, **Next.js**, **PostgreSQL**, and **Docker**.

#### 🤖 Automation & Integrations
*   **[vibe-bot-project](https://github.com/bysharad/vibe-bot-project)** — A local-first Telegram AI companion powered offline by Hugging Face Transformers and Ollama running on your hardware. Written in **Python** with **PyTorch** and **Httpx**.
*   **[ghostnet-bot](https://github.com/bysharad/ghostnet-bot)** — A lightweight Telegram search bot that scrapes and formats DuckDuckGo web search results directly in chat, requiring no external API tokens. Built with **Python** and **BeautifulSoup**.
